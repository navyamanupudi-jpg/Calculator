# Calculator Project in C Language

## Description
This is a console-based calculator project developed using C language in Dev-C++ IDE. It performs basic arithmetic operations and allows continuous calculations using a loop.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Power (^ using math.h)
- Modulus (%)
- Square operation
- Continuous calculations using do-while loop
- Input validation for division and modulus by zero

## Technologies Used
- C Language
- Dev-C++ IDE
- GCC Compiler
- math.h library

## How It Works
1. Program displays calculator menu
2. User enters expression (example: 5 + 3)
3. Program performs selected operation using switch case
4. Displays result
5. Asks user if they want to continue (Y/N)

## Example Input/Output
Input:
5 + 3  
Output:
8

Input:
4 ^ 2  
Output:
16

## Error Handling
- Division by zero is handled
- Modulus by zero is handled
- Invalid operator gives error message

## Conclusion
This project helps understand the use of:
- switch case
- loops (do-while)
- functions from math library
- user input handling in C
